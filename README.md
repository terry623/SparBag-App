# SparBag
> 預約行李托運，輕鬆旅行

### Installation

```sh
$ npm install
$ npm run schema
$ npm run start
```

### Install Database

>可參考 [PostgreSQL安裝教學](https://youtu.be/djPMnhVa76Y)，將環境變數加入以下值

| 變數 | 值 |
| ------ | ------ |
| PG_DB_NAME | 創建的 DB 名稱 |
| PG_USERNAME | 安裝 DB 時設的帳號 |
| PG_PASSWORD | 安裝 DB 時設的密碼 |
| PG_PORT | 安裝 DB 時設的 Port |
| PG_HOSTNAME | localhost |
